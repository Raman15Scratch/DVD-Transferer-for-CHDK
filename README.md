# DVD Transferer for CHDK
This is my DVD Transferer for Canon Cameras with CHDK! With this tool, you can literally watch DVD discs on a Canon Camera with CHDK and USB Mini B!

Importing Captions is optional btw.

Download: Select the folder of the version you want to download in the repository files, you can download main.exe or enter the source code folder in the folder of the version you want to download the source code of that version. To download the latest version, check releases.
